## 版本回退
---
### 未提交
* 修改了工作区代码，但是没有`git add `提交到暂存区
* 执行`git add`但是还没有`git commit`提交本地仓库

```
# 相当于清空工作区和暂存区的代码
git reset --hard
```

### 已提交但未推送
* 代码已经`git commit`到本地仓库但还没有`git push`到远程仓库

1. 回退到上个版本
```
git reset --hard HEAD^
```
2. 回退多个版本
```
git reset --hard HEAD~2
```
3. 回退指定版本
```
git reset --hard <commit_id>
```
4. 回退到远程仓库最新版本 
```
git reset --hard origin/master
```
### 已推送
* 已经`git push`到远程仓库
