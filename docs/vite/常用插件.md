# `vue3`组件命名 `vite-plugin-vue-setup-extend`
* `vue3`使用`script setup`标签时组件命名需要多写一个标签
  ```html
  <script lang="ts">
    import { defineComponent } from 'vue'
    export default defineComponent({
      name: 'OrderList'
    })
  </script>
  ```
* 直接在一个标签 `<script lang="ts" setup name="OrderList">`
